# Pythagoras
harmony = relations between numbers

![aulos](aulos.ico)


## Installation

```sh
git clone https://github.com/filyp/pythagoras.git
cd pythagoras
pip3 install -r requirements.txt
```

## Running

```sh
python3 pythagoras/dashboard.py
```

For help, run:
```sh
python3 pythagoras/dashboard.py -h
```
